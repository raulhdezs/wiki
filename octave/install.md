How to install GNU Octave in Linux 
==================================

1. Add repository for Octave


```sh
    $ sudo apt-add-repository ppa:octave/stable
```

2. Retrieve new lists of packages

```sh
    $ sudo apt-get update
```

3. Install Octave

```sh
    $ sudo apt-get install octave
```

4. Launch Octave
```sh
    $ octave
```
